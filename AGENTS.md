### Доступные глобальные скиллы (Artex Skills)
В системе глобально установлены следующие скиллы (доступны для Gemini, Claude, Codex и Copilot):
- `project-intake` — Универсальный скилл для первичного анализа и погружения в проект.
- `plan-and-wait` — Планирование задач и ожидание подтверждения.
- `docs-closeout` — Завершение работы с документацией.
- `context7-check` — Проверка контекста проекта.
- `gh-actions-repair` — Диагностика и починка GitHub Actions через CLI.
- `review-code`, `readme-refresh`, `api-doc-sync`, `cleanup-pass`, `env-drift-check`, `db-change-safety` — Универсальные скиллы для ревью, документации и безопасности.

*(Установлено путем глобального физического копирования файлов конфигурации из artex-skills; для Copilot используется объединенный `~/.copilot/copilot-instructions.md`)*
